Kernel driver acbel-fsg032¶

Supported chips:

  • ACBEL FSG032-00xG power supply.

Author: Lakshmi Yadlapati <lakshmiy@us.ibm.com>

Description¶

This driver supports ACBEL FSG032-00xG Power Supply. This driver is a client to the core PMBus driver.

Usage Notes¶

This driver does not auto-detect devices. You will have to instantiate the devices explicitly. Please see How to instantiate I2C devices for details.
