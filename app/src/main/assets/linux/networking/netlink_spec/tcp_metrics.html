Summary¶

Management interface for TCP metrics.

Operations¶

get¶

Retrieve metrics.

attribute-set:

tcp-metrics

dont-validate:

[‘strict’, ‘dump’]

do:
request
attributes:

[addr-ipv4, addr-ipv6, saddr-ipv4, saddr-ipv6]

reply
attributes:

[addr-ipv4, addr-ipv6, saddr-ipv4, saddr-ipv6, age, vals, fopen-mss, fopen-syn-drops, fopen-syn-drop-ts, fopen-cookie]

dump:
reply
attributes:

[addr-ipv4, addr-ipv6, saddr-ipv4, saddr-ipv6, age, vals, fopen-mss, fopen-syn-drops, fopen-syn-drop-ts, fopen-cookie]

Delete metrics.

tcp-metrics

[‘strict’, ‘dump’]

[admin-perm]

request
attributes:

[addr-ipv4, addr-ipv6, saddr-ipv4, saddr-ipv6]
